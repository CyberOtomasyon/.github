# Cyber Otomasyon

**Tarım ve hayvancılık sektörü için akıllı otomasyon ve izleme çözümleri.**

Çiftlik ve kümes yönetiminden gerçek zamanlı sensör izlemeye, mobil uygulamalardan web panellerine kadar uçtan uca yazılım geliştiriyoruz.

---

## Ne Yapıyoruz?

| Alan | Açıklama |
|---|---|
| 📱 **Mobil Uygulama** | Flutter ile iOS & Android uygulamaları |
| 🌐 **Web Panel** | PHP & modern frontend ile yönetim panelleri |
| ⚙️ **IoT & Sensör** | Gerçek zamanlı cihaz izleme ve alarm sistemleri |
| ☁️ **Backend API** | .NET Core tabanlı mikroservis mimarisi |
